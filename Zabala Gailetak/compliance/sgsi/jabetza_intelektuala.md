# Jabetza Intelektualaren Politika

**Helburua:** Zabala Gailetak-en eta hirugarrenen jabetza intelektuala babestea.

## 1. Software Lizentziak

### 1.1 Erabilera Arauak
- Lizentziapeko softwarea bakarrik erabili daiteke.
- Erabat debekatuta dago software pirata edo "crack"-eatua instalatzea.
- Software bat behar bada, IT Sailari eskatu behar zaio lizentzia zuzena lortzeko.

### 1.2 Inbentarioa eta Auditoria
- Urtero inbentarioa egingo da lizentziak egiaztatzeko.
- Software baimenik gabekoa aurkitzen bada, berehala desinstalatu behar da.
- Langileak lizentzia urraketen arduradun dira beren ekipoetan.

## 2. Kode Iturria

### 2.1 Enpresaren Jabetza
- Enpresan garatutako kode guztia Zabala Gailetak-en jabetza da.
- Langileek ezin dute koderik eraman alde egitean.

### 2.2 Kodearen Publikazioa
- Ez da kodea biltegi publikoetara (GitHub Public, GitLab Public) igo behar baimenik gabe.
- Open Source proiektuetan parte hartzeko, CISO eta Zuzendaritzaren baimena behar da.

### 2.3 Erabilera Pertsonala
- Langileen denbora pertsonalean garatutako koderik ez da enpresaren jabetzakoa, baldin eta ez bada enpresaren baliabideak edo informazioa erabili.

## 3. Hirugarrenen Eskubideak

### 3.1 Copyright Material
- Ez erabili copyright bidez babestutako irudi, testu edo materialik baimenik gabe.
- Stock fotoen lizentziak egiaztatu eta errespetatu.
- Argazkiak eta edukia sortzen denean, jatorri lizentziak egokiak direla bermatu.

### 3.2 Open Source Liburutegiak
- Open Source liburutegiak erabiltzean, lizentziaren baldintzak (MIT, GPL, Apache, adibidez) errespetatu.
- GPL lizentziapeko liburutegiak erabiltzean, kontu izan gure kodea ere GPL bihurtu behar dela.
- IT Sailak Open Source lizentzien auditoriak egingo ditu.

## 4. Sekretu Industriala

### 4.1 Errezetak eta Formulak
- Elikagaien errezetak eta formula kimikoak sekretu gisa tratatuko dira (KONFIDENTZIALA maila).
- Errezeta nagusiak gune seguruan gordeko dira (fisikoki eta digitalki).

### 4.2 Prozesu Industrialak
- Produkzio prozesuak eta makina konfigurazioak babestuta daude.
- Langileek konfidentzialtasun hitzarmenak (NDA) sinatu behar dituzte errezetak edo prozesuak ezagutzen badituzte.

## 5. Patenteak eta Marken Erregistroa

### 5.1 Patenteak
- Enpresak asmakuntzak patentatzeaz arduratuko da beharrezkoa denean.
- Langileek asmakuntzen berri eman behar diote enpresari deskubritzen dutenean.

### 5.2 Marka Komertziala
- Zabala Gailetak-en marka eta logotipoa babestuta daude.
- Ez erabili enpresaren marka edo logotiporik baimenik gabe kanpoko jardueretan.

## 6. Konfidentzialtasun Hitzarmenak (NDA)

### 6.1 Barne Langileak
- Langile guztiek konfidentzialtasun klausula sinatu behar dute kontratu batean.
- Klausula hau lanpostua amaitu ondoren ere indarrean jarraitzen du.

### 6.2 Kanpoko Kolaboratzaileak
- Azpikontratistek, aholkulariek eta bestelako kolaboratzaileek NDA sinatu behar dute informazio sentikorra jasotzen badute.

## 7. Erantzukizunak
- **Langile guztiek:** Jabetza intelektuala errespetatu eta babestu.
- **IT Sailak:** Software lizentziak eta Open Source betetze-mailak kontrolatu.
- **CISO:** Informazio sentikorren babesa gainbegiratu.
- **Legezko Aholkulariak:** Jabetza intelektualaren urraketen kasuan aholkuak eman.

## 8. Arau-hausteak

### 8.1 Zigorrak
- Software pirateratua erabiltzea: Kaleratzea eta zigor ekonomikoa.
- Kode fuga edo sekretuen filtrazioa: Kaleratzea eta aurka ekintzak legalak.
- Copyright urraketa: Diziplina prozedura eta konponketa ekonomikoa.
