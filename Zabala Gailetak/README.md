# Zabala Gailetak

## Proiektuaren Deskribapena

"Zabala Gailetak" proiektuaren helburua enpresaren azpiegitura informatikoa modernizatzea
eta segurtasuna indartzea da. Hau lortzeko, webgune bat eta mugikorrerako aplikazio bat
garatuko dira, eta segurtasun neurri zorrotzak ezarriko dira.

## Egitura

- **src/**: Aplikazioaren iturburu kodea.
  - **web/**: Web aplikazioaren kodea.
  - **mobile/**: Mugikorreko aplikazioaren kodea.
  - **api/**: Backend API zerbitzuak.
- **infrastructure/**: Azpiegitura eta sareen konfigurazioak.
  - **network/**: Sareen diseinua eta konfigurazio fitxategiak.
  - **systems/**: Sistemen administrazioa eta konfigurazioak.
  - **ot/**: Operazio Teknologia (OT) eta industria-sareak.
- **security/**: Segurtasunari lotutako dokumentazioa eta tresnak.
  - **incidents/**: Intzidentzien erantzun planak eta erregistroak.
  - **forensics/**: Auzitegi-analisi informatikoa.
  - **audits/**: Hacking etikoa eta segurtasun auditoriak.
  - **siem/**: SIEM konfigurazioak eta arauak.
- **compliance/**: Araudi eta betetze legala.
  - **gdpr/**: Datuen babeserako dokumentazioa.
  - **sgsi/**: Segurtasunaren Kudeaketa Sistema (SGSI).
- **devops/**: DevOps eta automatizazioa.
  - **ci-cd/**: Etengabeko integrazioa eta banaketa (CI/CD).
  - **deployment/**: Hedapen scriptak eta konfigurazioak.
- **docs/**: Proiektuaren dokumentazio orokorra.
- **tests/**: Proba automatizatuak.

## Hasiera

Proiektua hasteko, irakurri `docs/` karpetako dokumentazioa.
